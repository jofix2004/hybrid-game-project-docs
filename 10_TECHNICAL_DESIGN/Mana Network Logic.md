# Mana Network Logic

Tài liệu này mô tả logic kỹ thuật để hiện thực mạng lưới mana.

## Trạng thái

Chờ hoàn thiện. Cần có sơ đồ node, quy tắc cập nhật và xử lý sự cố.

## Khung ban đầu

`Source -> Storage -> Transmission -> Consumer`

Hệ thống cần hỗ trợ dò kết nối, tính tiêu hao và phát hiện quá tải.
