# Dungeon Generation

Tài liệu này mô tả cách sinh hầm ngục theo từng tầng.

## Trạng thái

Chờ hoàn thiện. Cần xác định tầng nào dùng hang tự nhiên, tầng nào dùng phòng-hành lang.

## Khung ban đầu

- Sinh theo seed và chỉ số tầng.
- Có checkpoint định kỳ.
- Kết hợp nhiều thuật toán để tạo cảm giác đa dạng.
